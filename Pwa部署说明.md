# 🚀 手机主屏幕 PWA 部署说明

您下载了所有文件后，只需要按照以下步骤，即可将此工具变成手机里的原生 App（免安装、无广告、可离线使用）。

## 准备工作
请将您刚才下载的所有文件放在同一个文件夹内：
- `index.html` (网页主体，已将您原版代码全部嵌入，并集成了 PWA 功能)
- `manifest.json` (App配置文件)
- `sw.js` (离线缓存服务)
- `icon-192.png` / `icon-512.png` / `icon-180.png` / `favicon-32.png` (为您自动生成的应用图标)

## 如何将其变成手机 App？

由于浏览器的安全限制，**PWA 必须运行在 HTTPS 环境下**，不能直接通过文件管理器打开运行。

### 方案一：使用免费的 Vercel 静态托管（推荐，最简单）
1. 打开浏览器访问：[vercel.com](https://vercel.com/) 并注册（可直接用 Github 账号登录）。
2. 在控制台选择 **"Add New" -> "Project"**。
3. 如果您不会用 Git，可以寻找类似 **Netlify** 的服务（支持直接拖拽整个文件夹上传部署）。
4. 部署完成后，系统会给您分配一个 HTTPS 链接（如 `xxx.vercel.app`）。

### 方案二：使用 GitHub Pages
1. 在 GitHub 上创建一个新仓库。
2. 将这 7 个文件直接上传到仓库根目录并提交。
3. 在仓库的 **Settings -> Pages** 中，将 Source 选为 `main` 分支。
4. 等待 1 分钟后，访问 `https://您的用户名.github.io/仓库名/`。

### 方案三：使用本地局域网（适合仅在自己家里电脑和手机之间用）
1. 在电脑上运行一个本地服务器，例如在装有 Python 的电脑上打开终端，进入该文件夹，运行 `python -m http.server 8080`。
2. 手机连上同一个 WiFi，在手机浏览器中输入电脑的局域网 IP（如 `http://192.168.1.100:8080`）。

## 添加到主屏幕
当您通过上述任意 HTTPS（或本地 IP）方式在手机浏览器中打开网页后：
- **苹果系统 (Safari)**：点击底部中间的“分享”图标 ⬆️，向上滑动找到并点击 **“添加到主屏幕”**。
- **安卓系统 (Chrome/Edge)**：页面顶部会自动弹出“📲 点击将‘推算系统’安装到桌面”横幅，点击确认即可。如果没有，您也可以点击浏览器右上角菜单栏 (⋮) 的 **“添加到主屏幕”**。

完成后，返回手机桌面，即可像打开原生 App 一样，秒开全屏运行您的推算系统了！
